<h1 align="center">Jarfix</h1>
<p align="center">A program that fixes <code>.jar</code> file associations on Windows.</p>

<p align="center">
  <img src="https://img.shields.io/github/v/release/qMaxXen/Jarfix?logo=github" alt="GitHub release"/>
  <img src="https://img.shields.io/github/downloads/qMaxXen/Jarfix/total?logo=github" alt="GitHub downloads"/>
  <img src="https://img.shields.io/github/license/qMaxXen/Jarfix?logo=github" alt="License"/>
</p>

Jarfix is a program that fixes the `.jar` file association so Java applications can run correctly. When you double-click a `.jar` file, Windows must know which Java runtime to use.

This program is similar to [Johann LÃ¶fflmann's original Jarfix](https://johann.loefflmann.net/en/software/jarfix/index.html), but it associates `.jar` files with only 64-bit Java runtime versions 17 or higher. If no suitable version is found, it will download and install Java 21, then set `.jar` files to use that Java runtime.

## Download
Go to the [releases](https://github.com/qMaxXen/Jarfix/releases/latest) section and download `Jarfix.exe`.
